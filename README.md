# FYP-3

AI Fitness Trainer — Final Year Project

## Project Structure

| Folder | Description | Tech Stack |
|--------|-------------|------------|
| `AI-Fintess-Trainer-Frontend-V3-main` | Frontend (React + Vite) | TypeScript, React, Tailwind, Supabase |
| `AI-Fitness-Trainer-Exercises-main` | Exercise Detection API | FastAPI, MediaPipe, OpenCV |
| `AI-Workout-Suggestion-main` | Workout Suggestion API | FastAPI, Google Gemini AI |
| `FYP-Assistent--main` | FitBot Chat Assistant API | FastAPI, Groq AI |

## Deployment

All services are deployed on [Render](https://render.com) from this monorepo.
Each service specifies its `rootDir` in its `render.yaml`.
